VGMToolbox – Manual

VGMToolbox (VGMT) is an application to aid in the auditing of yourVideo Game Music (VGM) collections. Inspired by rom organizers likeclrMAMEPro and the GoodTools applications, VGMT aims to offer thebest of both tools while focusing on VGM related file types (NSF,PSF, etc...). The key feature of the VGMT is the ability tocalculate checksums over the actual data of each of these formats. VGMT also offers various tools that aid the VGM collector, datafile creator, and music ripper.

Datafile Creator


Tools

[Hoot] XML Builder


Description: The Hoot XML Builder generates skeleton XML files for the GBS/NSF file formats. It can be useful for batch creation of Hoot xml files for GBS/NSF files. XML file(s) will be output in the "hoot" subdirectory of the application folder.

Usage: Drag and Drop files or folders onto the text box in the upper left.

Options: (At least on must be checked for output.)

  • Combine output to single XML file: All XML will be output to a single file (combined).
  • Output one file per game: A single XML file will be generated for each input game.

[NSFE] NSFE --> NSF + M3U


Description: The NSFE to .m3u converter takes an NSFE format file and creates an NSF with an Nezplug format .m3u playlist. The files will be output in the same directory as the source file.

Usage: Drag and Drop files or folders onto the text box in the upper left.
